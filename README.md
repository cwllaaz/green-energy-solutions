# Green Energy Solutions

A professional 5-page corporate website demo for a renewable energy company.
This project is designed as an Upwork portfolio sample for small business and
service-company website work.

## Pages

- Home
- About
- Services
- Portfolio
- Contact

## Features

- Responsive desktop and mobile layout
- Commercial B2B visual style
- Hero section with real imagery
- Service cards and project case studies
- Team, metrics, and value sections
- Contact form UI with demo success state
- Basic page metadata and Open Graph setup

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react icons
- Local portfolio images in `public/images`

## Run Locally

```bash
npm install
npm run dev
```

Open (https://green-energy-solutions-beta.vercel.app/) in your browser.

## Build

```bash
npm run lint
npm run build
```

## Contact Form

The current contact form is frontend-only and shows a success message after
submit. For a real client website, connect it to one of these options:

- Formspree for the fastest email-based setup
- Resend with a Next.js route handler for custom email delivery
- Supabase for storing inquiries in a database

## Deployment

This project is ready to deploy on Vercel:

1. Push the project to GitHub.
2. Import the repository in Vercel.
3. Deploy with the default Next.js settings.
